# OCE Prototype (MVP v0.1)

**Purpose:** Walking skeleton for the OCE Core + Router + 3 base modules (Strategy/Structure/Risk) + MetaCore stubs (CFL-Ethics, BiasSentinel, EvidenceEngine, GPS, SafetyGate) + MC-lite memory + simple validator.

## Quick start
```bash
python -m oce.demo
```

This runs a small end-to-end flow: Core → Router → Modules → MetaCore → Validator → Memory.
